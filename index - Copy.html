About Me

I'm an engineer based in Lienz (Osttirol) and Villach (Kärnten), Austria, working as an independent freelancer on projects that bridge hardware and software. My background covers desktop and embedded application development with Qt/QML and C++, EV charging infrastructure built on the Open Charge Point Protocol (OCPP), and industrial automation engineering using Siemens TIA Portal / PLCs. I also build scripts, tools and web applications with Python and JavaScript. I work with clients remotely across Austria and the DACH region, and on-site in Osttirol and Kärnten.

Services

Qt / C++ / QML Applications

Cross-platform desktop and embedded software with hardware integration — HMIs, device control panels and data acquisition tools built on Qt and modern C++.

EV Charging Systems (OCPP)

Charge point firmware and backend integration using the Open Charge Point Protocol — from charger communication to central management system connectivity.

PLC & Industrial Automation

Automation engineering with Siemens TIA Portal — PLC programming, HMI visualization and commissioning for industrial control systems.

Custom Software (Python / JavaScript)

Scripts, automation tools, data processing pipelines and web applications tailored to your workflow.
